# 💰 Expense Tracker

A simple Expense Tracker web application built using Python and Streamlit.

---

## 🚀 Features

- Add expenses with category and description
- Store expense data locally
- View all expenses
- Simple and clean UI
- Lightweight and easy to use

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🌐
- Pandas 📊
- JSON (data storage)
